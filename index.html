Training Wheels

Welcome to Training Wheels.

If you are a student and are looking for your environment,
please ensure you're using the exact web address that has been
provided by an instructor. It should be something like:

http://name.course.training.com/

If you are an instructor and are looking for the controller
interface, please check you have the correct address and port
number. The controller will normally be on port 8000, so:

http://training.com:8000/
